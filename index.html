Passos para hosperdar um projeto no Github Pages

  1. Crie um repositorio no GIthub com o mesmo nome do seu projeto

  2. No Terminal, inicialize o repositorio local com Git init

  3. Altere o nome da branch principal para main

  4. adicione os arquivos com git add . e faça o commit com git commit -m "first commit"

  5. conecte ao repositorio remoto com:
    git remote add origin https:

  6. envie os arquivos com git push -u origin main

  7. No Github, vá em settings > pages

  8. em "branch", selecione mais e a pasta/ (root) clique no save

  9. alguns segundos
